===================
Edgengram Tokenizer
===================

A tokenizer of type **edgeNGram**.


The following are settings that can be set for a **edgeNGram** tokenizer type:


==============  ===============================
 Setting         Description                   
==============  ===============================
**min_gram**    Defaults to **1**.             
**max_gram**    Defaults to **2**.             
**side**        Either **front** or **back**.  
==============  ===============================
