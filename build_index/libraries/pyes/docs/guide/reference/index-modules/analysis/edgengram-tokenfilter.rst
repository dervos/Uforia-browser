=====================
Edgengram Tokenfilter
=====================

A token filter of type **edgeNGram**.


The following are settings that can be set for a **edgeNGram** token filter type:


==============  ===============================
 Setting         Description                   
==============  ===============================
**min_gram**    Defaults to **1**.             
**max_gram**    Defaults to **2**.             
**side**        Either **front** or **back**.  
==============  ===============================
